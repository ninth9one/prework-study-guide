HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Section elements are semantic elements that define a section in a document containing thematic content
  • HTML uses the 'id' attribute as a way to identify elements and select them

CSS

  • 'CSS Rule' a selector defines the element or attributes to which the rules or, declarations will apply
  • 'CSS Rule' a declaration contains two important components: the CSS property we want to apply and the value property
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A command is used to commit all files in the working branch
  • git commit -m flag to associate a message with our commit

Javascript

  • Function is followed by empty parentheses after the name
  • cA varialbe is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • 'Math.floor()' and 'Math.random()' are both properties of js' built-in math object that can be used together to generate a random number
